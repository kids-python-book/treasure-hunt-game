## Treasure Hunt Game (Book Companion)

This repo is the companion project for **The Lost Treasure of Python Island**.

It contains the full, runnable code for the Treasure Hunt game built throughout the book. The book keeps code snippets short (mobile-friendly), and this repo holds the complete project so you can run it without copy/paste.

### What's inside

- `main.py`: starts the game
- `game/`: the game modules (player, monsters, rooms, encounters)
- `docs/`: extra notes (optional)

## Download the repo

You can download the code in two common ways.

### Option A: Download with Git (recommended)

1. Open a terminal.
2. Clone the repo:

```bash
git clone <REPLACE_WITH_YOUR_GITHUB_REPO_URL>
```

3. Enter the folder:

```bash
cd treasure-hunt-game
```

### Option B: Download as a ZIP

1. Open the repo on GitHub.
2. Click **Code** → **Download ZIP**.
3. Unzip it.
4. Open a terminal and `cd` into the unzipped folder (it will usually be named something like `treasure-hunt-game-main`).

## Run the game

### Step 1: Make sure Python is installed

In a terminal, run:

```bash
python3 --version
```

If that doesn't work on your computer, try:

```bash
python --version
```

### Step 2: Start the game

From inside the repo folder, run:

```bash
python3 main.py
```

If your system uses `python` instead of `python3`, run:

```bash
python main.py
```
